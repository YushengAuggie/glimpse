flowchart LR
  U([Client]) -->|1 · upload| API[Upload API]
  API -->|store original| S3[(Object store)]
  API -->|enqueue| Q[[Queue]] --> W[Resizer worker]
  W -->|write sizes| S3
  W -->|mark done| DB[(Jobs DB)]
  S3 --> CDN[(CDN)] -->|2 · serve| U

What each node is

Upload API

Validates, stores the original, enqueues one resize job, returns a job id.

Queue + worker

Workers pull jobs, generate each size, retry with backoff, flip status.

Storage + CDN

One bucket, predictable keys (img/{id}/{size}.webp); CDN caches by key.
